# GPT Language Model From Scratch (PyTorch)

This project implements a GPT-style Transformer trained on the OpenWebText dataset to generate natural language text.

The model learns language patterns directly from raw internet text and produces coherent paragraphs without instruction tuning.

## Features
- Transformer architecture implemented from scratch
- Autoregressive text generation
- Top-K sampling decoding
- GPU training using PyTorch

## Example
Prompt:
Q: What is artificial intelligence?
A:

Output:
Artificial intelligence is a system designed to perform tasks that typically require human reasoning...

## Usage
Open the notebook and run the inference section after loading the trained weights.
